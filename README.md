# sagar-wants-to-make-ios-brainrot-apps

Expert Swift & iOS development skill for AI coding agents. Covers SwiftUI architecture, design systems, networking, state management, persistence, testing, and production patterns.

## Install

```bash
npx skills add adisinghstudent/swift-ios
```

## What's Included

- **Project structure** — Feature-based MVVM organization
- **Design system** — Colors, typography, spacing, reusable components (Tailwind-to-SwiftUI mapping)
- **Architecture** — MVVM with @Observable (iOS 17+) and ObservableObject patterns
- **Networking** — Async/await API client with SSE streaming
- **Navigation** — NavigationStack, TabView, sheets/modals
- **State management** — @Observable, @Environment, Combine
- **Persistence** — SwiftData, UserDefaults/@AppStorage, Keychain
- **Authentication** — Supabase Auth integration pattern
- **UI patterns** — Chat interface, pull-to-refresh, infinite scroll, empty states, search
- **Testing** — XCTest unit tests, SwiftUI previews
- **Package management** — SPM with common iOS packages
- **Swift 6 concurrency** — Sendable, @MainActor, structured concurrency
- **Xcode CLI** — Build, test, and manage projects from terminal
- **Quick reference** — Tailwind CSS to SwiftUI translation table

## Usage

After installation, the skill activates when you work with Swift/iOS code or mention iOS, SwiftUI, Xcode, etc.
